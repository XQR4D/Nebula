# 🌌 Nebula — Космическая Видео-Галерея

**Nebula** — это легкий и стильный веб-интерфейс для вашей библиотеки видеофайлов. Система автоматически подтягивает видео из ваших **GitHub Releases**, позволяя удобно просматривать их, скачивать или встраивать на другие сайты через Iframe.

## 🚀 Как это работает?

В отличие от обычных сайтов, Nebula не хранит тяжелые видео в самом репозитории. Это позволяет держать код чистым и быстрым.
1. Вы загружаете видео в раздел **Releases**.
2. Сайт через **GitHub API** находит все видео во всех релизах.
3. Видео отображаются в красивой "стеклянной" (Glassmorphism) сетке.

## 🛠 Установка и настройка

### 1. Клонирование
```bash
git clone [https://github.com/onenicked/Nebula.git](https://github.com/onenicked/Nebula.git)

### 2. Настройка под свой репозиторий
В файле script.js измените константы на свои:

JavaScript

const GITHUB_USER = "ТВОЙ_ЛОГИН";
const GITHUB_REPO = "ТВОЙ_РЕПОЗИТОРИЙ";

### 3. Как добавлять видео
Перейдите в раздел Releases вашего репозитория.

Нажмите "Draft a new release".

Создайте новый тег (например, v1.0.0).

В область Attach binaries перетащите ваши .mp4, .mov или .webm файлы.

Опубликуйте релиз. Сайт обновится автоматически!

💎 Особенности проекта
Glassmorphism UI: Современный дизайн с эффектом матового стекла.

Dynamic Routing: Каждое видео имеет свою уникальную ссылку.

Embed System: Готовый код вставки видео для других сайтов.

API Optimized: Минимум запросов, максимум скорости.
