## **Nebula** — это легкий 8-битный плеер, созданый при поддержке t.me/bipbupyoutube. 
Плеер автоматически подтягивает видео из **GitHub Releases**, позволяя удобно просматривать их, скачивать или встраивать на другие сайты через Iframe.
